# OLA Ride Booking Analysis | SQL & Power BI

## Project Overview

This project analyzes over 20,000 OLA ride bookings to uncover business insights related to ride demand, cancellations, revenue generation, customer satisfaction, and vehicle performance.

The analysis was performed using SQL for querying and data analysis, Excel for data cleaning, and Power BI for interactive dashboard creation.

---

## Objective

The primary objectives of this project are:

* Analyze ride booking patterns.
* Track successful and cancelled rides.
* Evaluate vehicle type performance.
* Measure customer and driver ratings.
* Monitor revenue and booking trends.
* Create an interactive dashboard for business decision-making.

## Dataset Information

The dataset contains 20,000+ OLA ride booking records, including:

* Booking Value
* Payment Method
* Customer Rating
* Driver Rating
